# Fisherman-Feedback
This Fisherman Feedback analysis is a compilation of scripts written by John Froeshke, Lisa Hollensead, and myself. 

The script gt_sentiment is LH original and updated version is Word Frequency_LE edited by myself.

The script workflow_automate_test is JF original and updated version is Map_Sentiment_LE edited by myself. 

The script Broad Sentiment Eguia is written by myself.

All JF contribution can be found here: https://github.com/jfroeschke/GrayTriggerfishFF


The order of scripts should be run as Broad Sentiment Eguia -> Word_Frequency_LE -> Map_Sentiment_LE

These scripts are developed to analyze Fisherman Feedback for different stocks managed by the Gulf Council.
